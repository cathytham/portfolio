Split Decision

Project detail page

SplitDecision is an interactive group decision-making web application that allows users to create teams, create polls, play decision games like wheel spinning, horse racing, and rock-paper-scissors, and share availability through a collaborative calendar. The project includes user research, low and high-fidelity prototypes, and a full implementation built with HTML, CSS, JavaScript, Python (Flask) and MongoDB.

Contribution

Developed the Spin the Wheel and Horse Racing decision game interfaces, including animations and interactive logic.
Designed and implemented the Availability Calendar, with features such as drag-to-select, edit/delete availability, hover highlights, and dynamic slot shading.
Built functionality to automatically consolidate multiple users’ availability.

Tools

Python(Flask), HTML, CSS, JavaScript.
