Bunny Character

Thesis
A playful 3D bunny character brought to life through animation and real-time interaction.

This project explores the creation of a stylized 3D bunny character, modeled in Blender and animated using Mixamo. The character was then integrated into Unreal Engine to create an interactive real-time experience, focusing on smooth animation, visual appeal, and character personality.

My exact contribution

Modeling and texturing the 3D bunny character in Blender.
Rigging and applying animations using Mixamo.
Importing and integrating the animated character into Unreal Engine.
Setting up lighting and scene composition for presentation.
Refining animations and optimizing the character for real-time rendering.
Ideation
