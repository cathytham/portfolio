Thesis
Finding animal shelters should be simple — this project transforms complex data into an accessible, interactive map experience.

This project is a full-stack MERN application designed to visualize a large dataset of animal shelters across North America. By integrating Mapbox Leaflet with an external API, the platform allows users to explore locations through an intuitive, map-based interface. Users can navigate freely, search by region, and interact with map points to access key information such as shelter name, address, phone number, and email. The project emphasizes usability and clarity, turning raw data into a tool that supports discovery and accessibility. Developed through pair programming, the workflow encouraged collaboration and continuous feedback, strengthening both technical implementation and problem-solving approaches.

Design directions explored

List-based directory

An early idea focused on displaying shelters in a traditional list view with filters. While informative, it made browsing locations feel disconnected from geography.

Interactive map ✦

The chosen direction centers on a map-based interface using Leaflet. Users can explore shelters spatially, click markers, and access detailed information in context.
