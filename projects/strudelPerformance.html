Thesis
A real-time audiovisual performance where code becomes both instrument and medium.

This project explores live coding as a form of performance, using Strudel for algorithmic sound generation and Hydra for real-time visuals. By writing and modifying code during the performance, sound and visuals evolve simultaneously, creating an improvised and reactive experience. The work focuses on rhythm, repetition, and glitch aesthetics, transforming code into an expressive and performative tool.

My exact contribution

Designed and performed live-coded sound using Strudel.
Developed algorithmic rhythms and evolving sound patterns.
Experimented with timing, layering, and repetition to shape the performance.
Iterated in real time to create dynamic and responsive audio output.
Ideation

Design directions explored

Structured composition

An initial idea focused on pre-composed sound sequences. While predictable and controlled, it reduced the spontaneity and responsiveness that live coding enables.

Live-coded improvisation ✦

The chosen direction embraces improvisation through live coding. Sound is generated and manipulated in real time, allowing rhythm, texture, and intensity to evolve dynamically during the performance.
