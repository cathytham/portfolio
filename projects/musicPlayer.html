Thesis
A simple interface can transform listening into an interactive experience.

This project is an interactive music player built using HTML, CSS, and JavaScript, focusing on simplicity, responsiveness, and user control. It allows users to play, pause, and navigate between tracks through a clean and intuitive interface. The goal was to explore how basic web technologies can be used to create engaging media interactions without relying on external frameworks. Emphasis was placed on smooth playback, clear visual feedback, and accessible controls, turning a minimal interface into a functional and enjoyable listening experience. Completed within a short timeframe, the project highlights efficient front-end development and attention to user interaction details.

Ideation

Design directions explored

Feature-heavy player

An initial direction explored adding advanced features such as playlists, volume controls, and visualizers. While powerful, it introduced unnecessary complexity for a short project and distracted from the core interaction.

Minimal player ✦

The chosen direction focuses on a clean, minimal interface with essential controls — play, pause, and track navigation. This approach prioritizes usability, clarity, and smooth interaction while showcasing how simple web technologies can create an engaging audio experience.
